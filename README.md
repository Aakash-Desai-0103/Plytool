# C++ Syntax Verification using PLY

A simple academic project that performs lexical and syntax analysis for a subset of C++ constructs using Python Lex-Yacc (PLY).

## Supported Constructs
- Simple data-type declarations  
- Array declarations  
- Function declarations (prototypes)  
- Function definitions  
- Ifâ€“Else statements  

Each construct is parsed using a separate parser with a shared lexer.

## Tech Used
- Python  
- PLY (Python Lex-Yacc)  
- C++ (subset)
